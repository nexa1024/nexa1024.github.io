DSA-Day09-42.接雨水

<!--more-->

## 题目

给定 n 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子下雨之后能接多少雨水。

**示例 1：**

```
输入：height = [0,1,0,2,1,0,1,3,2,1,2,1]
输出：6
解释：上面是由数组 [0,1,0,2,1,0,1,3,2,1,2,1] 表示的高度图，在这种情况下，可以接 6 个单位的雨水（蓝色部分表示雨水）。
```

**示例 2：**

```
输入：height = [4,2,0,3,2,5]
输出：9
```

**提示：**

- `n == height.length`
- `1 <= n <= 2 * 10^4`
- `0 <= height[i] <= 10^5`

## 标签

数组、双指针、动态规划、栈、单调栈

# 🚀 30秒速记卡片

## 一句话口诀

> 双指针向中靠，矮边先结算，更新最大高

## 核心代码（伪代码）

```plain
left = 0, right = n-1
leftMax = 0, rightMax = 0
result = 0

当 left < right 时:
    若 height[left] < height[right]:
        leftMax = max(leftMax, height[left])
        result += leftMax - height[left]
        left++
    否则:
        rightMax = max(rightMax, height[right])
        result += rightMax - height[right]
        right--
```

## 记忆要点

1. **核心公式**：每个位置接水量 = `min(左侧最高, 右侧最高) - 当前高度`
2. **双指针精髓**：哪边高度限制更低，就先结算哪边
3. **时间复杂度**：O(n)，只需遍历一次
4. **空间复杂度**：O(1)，只用了几个变量
5. **易错点/注意事项**：
   - 数组长度至少为 3 才能接水（两边无法存水）
   - 初始时 leftMax 和 rightMax 都为 0
   - 每次只移动一个指针（left 或 right）
   - 移动前先更新对应侧的最大高度

**核心思想解释：**

当 `height[left] < height[right]` 时，说明左侧是"短板"，此时 `leftMax` 一定是 `min(leftMax, rightMax)`，所以可以直接结算 left 位置的接水量。

同理，当 `height[left] >= height[right]` 时，右侧是"短板"，可以结算 right 位置。

# 💻 完整代码

## Java

```java
/**
 * 42. 接雨水 - 双指针解法
 * 时间复杂度：O(n)，空间复杂度：O(1)
 */
class Solution {
    public int trap(int[] height) {
        // 边界条件：至少需要3根柱子才能接水
        if (height == null || height.length < 3) {
            return 0;
        }

        int left = 0, right = height.length - 1;
        int leftMax = 0, rightMax = 0;
        int result = 0;

        // 双指针向中间靠拢
        while (left < right) {
            // 哪边高度更小，先结算哪边
            if (height[left] < height[right]) {
                // 更新左侧最大高度
                leftMax = Math.max(leftMax, height[left]);
                // 当前位置能接的水 = 左侧最大 - 当前高度
                result += leftMax - height[left];
                left++;
            } else {
                // 更新右侧最大高度
                rightMax = Math.max(rightMax, height[right]);
                // 当前位置能接的水 = 右侧最大 - 当前高度
                result += rightMax - height[right];
                right--;
            }
        }

        return result;
    }
}
```

## 📚 其他解法补充

### 动态规划解法

```java
class Solution {
    public int trap(int[] height) {
        if (height == null || height.length < 3) return 0;

        int n = height.length;
        int[] leftMax = new int[n];
        int[] rightMax = new int[n];

        // 预处理左侧最大值
        leftMax[0] = height[0];
        for (int i = 1; i < n; i++) {
            leftMax[i] = Math.max(leftMax[i - 1], height[i]);
        }

        // 预处理右侧最大值
        rightMax[n - 1] = height[n - 1];
        for (int i = n - 2; i >= 0; i--) {
            rightMax[i] = Math.max(rightMax[i + 1], height[i]);
        }

        // 计算接水量
        int result = 0;
        for (int i = 1; i < n - 1; i++) {
            result += Math.min(leftMax[i], rightMax[i]) - height[i];
        }

        return result;
    }
}
```
