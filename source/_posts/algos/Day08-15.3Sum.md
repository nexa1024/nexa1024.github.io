DSA-Day08-15.三数之和

<!--more-->

## 题目

给你一个整数数组 `nums`，判断是否存在三元组 `[nums[i], nums[j], nums[k]]` 满足 `i != j`、`i != k` 且 `j != k`，同时还满足 `nums[i] + nums[j] + nums[k] == 0`。请你返回所有和为 `0` 且不重复的三元组。

**注意**：答案中不可以包含重复的三元组。

**示例 1：**

```
输入：nums = [-1,0,1,2,-1,-4]
输出：[[-1,-1,2],[-1,0,1]]
```

**示例 2：**

```
输入：nums = [0,1,1]
输出：[]
```

**示例 3：**

```
输入：nums = [0,0,0]
输出：[[0,0,0]]
```

**提示：**

- `3 <= nums.length <= 3000`
- `-10⁵ <= nums[i] <= 10⁵`

## 标签

数组、双指针、排序

# 🚀 30秒速记卡片

## 一句话口诀

**排序 → 固定一个数 → 双指针对撞 → 三处去重**

## 核心代码（伪代码）

```plain
排序 nums

for i = 0 to n-3:
    # 去重①：第一个数重复跳过
    if i > 0 and nums[i] == nums[i-1]:
        continue

    left = i + 1
    right = n - 1

    while left < right:
        sum = nums[i] + nums[left] + nums[right]

        if sum == 0:
            记录结果

            # 去重②：第二个数去重
            while left < right and nums[left] == nums[left+1]:
                left++

            # 去重③：第三个数去重
            while left < right and nums[right] == nums[right-1]:
                right--

            left++
            right--

        else if sum < 0:
            left++    # 和太小，需要更大的数

        else:
            right--   # 和太大，需要更小的数
```

## 记忆要点

1. **排序是前提**：没有排序就没有双指针

2. **三处去重**：

   去重①：外层循环 `nums[i] == nums[i-1]`

   去重②：找到答案后 `left` 跳过重复

   去重③：找到答案后 `right` 跳过重复

3. **双指针移动**：`sum < 0` 左指针右移，`sum > 0` 右指针左移

4. **时间 O(n²)，空间 O(1)**

5. **与两数之和的区别**：本题必须排序（因为要返回值且去重）

# 💻 完整代码

## Java

```plain
class Solution {
    public List<List<Integer>> threeSum(int[] nums) {
        List<List<Integer>> result = new ArrayList<>();

        // 1. 先排序
        Arrays.sort(nums);

        // 2. 外层循环：固定第一个数
        for (int i = 0; i < nums.length - 2; i++) {
            // 去重①：如果第一个数重复，跳过
            if (i > 0 && nums[i] == nums[i - 1]) {
                continue;
            }

            // 【进阶优化】如果最小的三个数之和都大于0，后面不可能有解
            if (nums[i] + nums[i + 1] + nums[i + 2] > 0) {
                break;
            }

            // 【进阶优化】如果当前数加上最大的两个数都小于0，当前数太小，跳过
            if (nums[i] + nums[nums.length - 1] + nums[nums.length - 2] < 0) {
                continue;
            }

            // 3. 双指针查找另外两个数
            int left = i + 1;
            int right = nums.length - 1;

            while (left < right) {
                int sum = nums[i] + nums[left] + nums[right];

                if (sum == 0) {
                    // 找到一个有效三元组
                    result.add(Arrays.asList(nums[i], nums[left], nums[right]));

                    // 去重②：跳过第二个数的重复值
                    while (left < right && nums[left] == nums[left + 1]) {
                        left++;
                    }

                    // 去重③：跳过第三个数的重复值
                    while (left < right && nums[right] == nums[right - 1]) {
                        right--;
                    }

                    // 同时移动指针，继续查找
                    left++;
                    right--;
                } else if (sum < 0) {
                    // 和太小，需要更大的数，左指针右移
                    left++;
                } else {
                    // 和太大，需要更小的数，右指针左移
                    right--;
                }
            }
        }

        return result;
    }
}
```

## Kotlin

```plain
class Solution {
    fun threeSum(nums: IntArray): List<List<Int>> {
        val result = mutableListOf<List<Int>>()

        // 1. 先排序
        nums.sort()

        // 2. 外层循环：固定第一个数
        for (i in 0 until nums.size - 2) {
            // 去重①：如果第一个数重复，跳过
            if (i > 0 && nums[i] == nums[i - 1]) {
                continue
            }

            // 【进阶优化】
            if (nums[i] + nums[i + 1] + nums[i + 2] > 0) break

            if (nums[i] + nums[nums.size - 1] + nums[nums.size - 2] < 0) continue

            // 3. 双指针查找
            var left = i + 1
            var right = nums.size - 1

            while (left < right) {
                val sum = nums[i] + nums[left] + nums[right]

                when {
                    sum == 0 -> {
                        result.add(listOf(nums[i], nums[left], nums[right]))

                        // 去重②③
                        while (left < right && nums[left] == nums[left + 1]) left++

                        while (left < right && nums[right] == nums[right - 1]) right--

                        left++
                        right--
                    }
                    sum < 0 -> left++
                    else -> right--
                }
            }
        }

        return result
    }
}
```
