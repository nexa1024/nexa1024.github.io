## 题目概述
**HashSet 基础应用**。本题是 LeetCode 简单题，核心考点是**使用 HashSet 判断数组中是否存在重复元素**，是哈希表入门的第一道题。

## 为什么这道题重要
这是 **哈希表去重**的基础操作，掌握后可解决：
- [217. 存在重复元素](https://leetcode.cn/problems/contains-duplicate/)（本题）
- [219. 存在重复元素 II](https://leetcode.cn/problems/contains-duplicate-ii/)
- [220. 存在重复元素 III](https://leetcode.cn/problems/contains-duplicate-iii/)

## 解法对比
| 解法 | 时间复杂度 | 空间复杂度 | 说明 |
|------|-----------|-----------|------|
| 暴力枚举 | O(n²) | O(1) | 双重循环，性能差 |
| 排序后比较 | O(n log n) | O(1) 或 O(n) | 需要修改原数组或额外空间 |
| **HashSet** | **O(n)** | **O(n)** | **最优解，推荐背诵** |

**背诵重点**：HashSet 解法（O(n) 时间，一次遍历）

## 背诵模型
**模型/状态**

+ 我用一个哈希seen记录已经出现过的元素。

**不变量/约束**

+ 遍历到第 i 个元素时，seen 包含了前 i-1 个元素出现过的所有值

**推进/转移**

+ arr[i]没有出现在seen中

**终止/答案+边界**

+ 终止：当arr[i]出现在seen中。
+ 边界：空数组

**常见坑**

+ 忘记将数组元素add入哈希



## 代码
### java
```plain
class Solution {
    public boolean containsDuplicate(int[] nums) {
        HashSet<Number> seensSet = new HashSet<Number>();
        for(int x: nums) {
            if(seensSet.contains(x)) return true;
            seensSet.add(x);
        }
        return false;
    }
}
```

### kotlin
```plain
class Solution {
    fun containsDuplicate(nums: IntArray): Boolean {
        val seensSet = HashSet<Int>(nums.size)
        for(n: Int in nums) {
            if(seensSet.contains(n)) return true
            seensSet.add(n)
        }
        return false
    }
}
```
