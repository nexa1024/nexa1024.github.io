DSA-Day09-11.盛最多水的容器

<!--more-->

## 题目

给定一个长度为 `n` 的整数数组 `height`。有 `n` 条垂线，第 `i` 条线的两个端点是 `(i, 0)` 和 `(i, height[i])`。

找出其中的两条线，使得它们与 `x` 轴共同构成的容器可以容纳最多的水。

返回容器可以储存的最大水量。

**说明：** 你不能倾斜容器。

**示例 1：**

```
输入：[1,8,6,2,5,4,8,3,7]
输出：49
解释：图中垂直线代表输入数组 [1,8,6,2,5,4,8,3,7]。在此情况下，容器能够容纳水（表示为蓝色部分）的最大值为 49。
```

**示例 2：**

```
输入：[1,1]
输出：1
```

**提示：**

- `n == height.length`
- `2 <= n <= 10⁵`
- `0 <= height[i] <= 10⁴`

## 标签

数组、双指针、贪心

# 🚀 30秒速记卡片

## 一句话口诀

**双指针首尾对撞 → 移动较小指针 → 记录最大面积**

## 核心代码（伪代码）

```plain
left = 0
right = n - 1
maxArea = 0

while left < right:
    # 计算当前面积：矮柱子高度 × 宽度
    area = min(height[left], height[right]) × (right - left)
    maxArea = max(maxArea, area)

    # 关键：移动较矮的指针
    if height[left] < height[right]:
        left++
    else:
        right--

return maxArea
```

## 记忆要点

1. **双指针位置**：`left = 0`，`right = n - 1`（首尾对撞）

2. **面积计算公式**：`area = min(height[left], height[right]) × (right - left)`

3. **核心贪心策略**：**每次移动高度较小的指针**

   为什么移动较小指针？
   - 容积受限于较矮的柱子
   - 移动较高指针 → 宽度减小，高度仍受限 → 面积一定变小
   - 移动较矮指针 → 可能找到更高的柱子 → 面积可能变大

4. **时间 O(n)，空间 O(1)**

5. **易错点**：
   - 不要用 `height[left] × height[right]`（那是矩形面积）
   - 必须取 `min` 的高度（水会从矮的边溢出）

**第三步：关键决策分析**

```
为什么移动较小的指针？

假设 left=1(height=8), right=8(height=7):
  当前面积 = min(8,7) × 7 = 49

  如果移动right(7→3):
    新面积 = min(8,3) × 6 = 18  ❌ 变小了

  但我们只能移动right！
  因为移动left(8→6)的话:
    新面积 = min(6,7) × 7 = 42  ❌ 一定变小（高度受限，宽度也减）

  所以：移动较小指针才有希望找到更大的面积！
```

# 💻 完整代码

## Java

```plain
class Solution {
    public int maxArea(int[] height) {
        int left = 0;
        int right = height.length - 1;
        int maxArea = 0;

        while (left < right) {
            // 计算当前面积：矮柱子高度 × 宽度
            int currentArea = Math.min(height[left], height[right])
                              * (right - left);
            maxArea = Math.max(maxArea, currentArea);

            // 核心策略：移动高度较小的指针
            if (height[left] < height[right]) {
                left++;
            } else {
                right--;
            }
        }

        return maxArea;
    }
}
```
