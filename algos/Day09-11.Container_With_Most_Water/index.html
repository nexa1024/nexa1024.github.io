DSA-Day09-11.盛最多水的容器

题目

给定一个长度为 n 的整数数组 height。有 n 条垂线，第 i 条线的两个端点是 (i, 0) 和 (i, height[i])。

找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。

返回容器可以储存的最大水量。

说明： 你不能倾斜容器。

示例 1：

示例 2：

提示：

  • n == height.length
  • 2 <= n <= 10⁵
  • 0 <= height[i] <= 10⁴

标签

数组、双指针、贪心

🚀 30秒速记卡片

一句话口诀

双指针首尾对撞 → 移动较小指针 → 记录最大面积

核心代码（伪代码）

记忆要点

  1. 双指针位置：left = 0，right = n - 1（首尾对撞）

  2. 面积计算公式：area = min(height[left], height[right]) × (right - left)

  3. 核心贪心策略：每次移动高度较小的指针

    为什么移动较小指针？

    • 容积受限于较矮的柱子
    • 移动较高指针 → 宽度减小，高度仍受限 → 面积一定变小
    • 移动较矮指针 → 可能找到更高的柱子 → 面积可能变大

  4. 时间 O(n)，空间 O(1)

  5. 易错点：

    • 不要用 height[left] × height[right]（那是矩形面积）
    • 必须取 min 的高度（水会从矮的边溢出）
