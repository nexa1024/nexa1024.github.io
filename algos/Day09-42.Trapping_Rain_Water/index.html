DSA-Day09-42.接雨水

题目

给定 n 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子下雨之后能接多少雨水。

示例 1：

示例 2：

提示：

  • n == height.length
  • 1 <= n <= 2 * 10^4
  • 0 <= height[i] <= 10^5

标签

数组、双指针、动态规划、栈、单调栈

🚀 30秒速记卡片

一句话口诀

双指针向中靠，矮边先结算，更新最大高

核心代码（伪代码）

记忆要点

  1. 核心公式：每个位置接水量 = min(左侧最高, 右侧最高) - 当前高度
  2. 双指针精髓：哪边高度限制更低，就先结算哪边
  3. 时间复杂度：O(n)，只需遍历一次
  4. 空间复杂度：O(1)，只用了几个变量
  5. 易错点/注意事项：
    • 数组长度至少为 3 才能接水（两边无法存水）
    • 初始时 leftMax 和 rightMax 都为 0
    • 每次只移动一个指针（left 或 right）
    • 移动前先更新对应侧的最大高度

核心思想解释：

当 height[left] < height[right] 时，说明左侧是”短板”，此时 leftMax 一定是 min(leftMax, rightMax)，所以可以直接结算 left 位置的接水量。

同理，当 height[left] >= height[right] 时，右侧是”短板”，可以结算 right 位置。
