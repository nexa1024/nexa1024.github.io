DSA-Day08-15.三数之和

题目

给你一个整数数组 nums，判断是否存在三元组 [nums[i], nums[j], nums[k]] 满足 i != j、i != k 且 j != k，同时还满足 nums[i] + nums[j] + nums[k] == 0。请你返回所有和为 0 且不重复的三元组。

注意：答案中不可以包含重复的三元组。

标签

数组、双指针、排序

🚀 30秒速记卡片

一句话口诀

排序 → 固定一个数 → 双指针对撞 → 三处去重

核心代码（伪代码）

记忆要点

  1. 排序是前提：没有排序就没有双指针

  2. 三处去重：

    去重①：外层循环 nums[i] == nums[i-1]

    去重②：找到答案后 left 跳过重复

    去重③：找到答案后 right 跳过重复

  3. 双指针移动：sum < 0 左指针右移，sum > 0 右指针左移

  4. 时间 O(n²)，空间 O(1)

  5. 与两数之和的区别：本题必须排序（因为要返回值且去重）
